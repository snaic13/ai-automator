# AI-Automator MVP

Автоматизация бизнес-процессов на базе GPT.

## Быстрый старт

1. Установи зависимости:
   ```bash
   pip install -r requirements.txt
   ```

2. Задай API ключ:
   ```bash
   export OPENAI_API_KEY="sk-..."
   ```

3. Запусти веб-интерфейс:
   ```bash
   python app.py
   ```

4. Открой http://localhost:5000

## Функции

- **Обработка документов** — извлечение информации из счетов, договоров
- **Поддержка клиентов** — автоматические ответы на вопросы
- **Генерация отчётов** — создание отчётов из данных

## Стоимость (GPT-5.4 mini)

- Input: $0.75 / 1M токенов
- Output: $4.50 / 1M токенов
- $10 хватит на ~1000 запросов
